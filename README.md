# Tabs Later
A Chrome extension to save and organize your tabs into lists.

## Development
**Tech Stack**
React, Redux, PouchDB for browser(IndexedDB)

**Local development**
Install the extension from local directory, and then start to watch local development changes.

    yarn install
    yarn start

**Make a build**

    yarn build
